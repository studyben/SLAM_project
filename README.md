# SLAM_project
### The code load sensors' data and generate map of robot.
#### Sensing data usage:
Encoder: 2 * n, left wheel, right wheel
FOG: 3 * n, delta angle in x, y, z
2D-Lidar: 286 * n, obstcle location in different directions
